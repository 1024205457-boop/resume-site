## 岗位匹配分析

### 强匹配项

**1. AI 应用落地经验 → "推动效果提升"**
- 猿辅导实习：设计 Prompt 策略接入智谱大模型，实现 63 条文案自动生产，人力从 3 人降至 0 人
- 这直接对应 JD 中"推动效果提升"和"AI 生成式能力"的要求

**2. 策略产品思维 → "定义理想体验，推动效果提升"**
- 猿辅导：设计转化漏斗（加微UV→领取UV→激活UV→购买）与流失分析模型
- 抖音黑客松获"优秀产品"奖，证明对抖音生态有理解

**3. 跨团队协作 → "与模型算法、工程技术团队合作"**
- 猿辅导：与业务侧梳理 4 大业务模块 15 个功能的操作流程
- 多次黑客松经历证明快速协作能力

**4. AI 技术栈 → "AI 生成式能力"**
- 掌握 Prompt Engineering、RAG、LangChain、Agent 开发
- 抖音黑客松中用 LangChain 重写后端，调试 Prompt

### 中等匹配项

**5. 内容生态理解**
- 抖音黑客松经历说明对抖音生态有接触，但非深度运营经验
- 猿辅导私域文案工作涉及内容生产，但场景不同

### 待补项

**6. 理工科背景**
- JD 注明"理工科优先"，你的背景是市场营销 + 人类学
- **应对策略**：用 AI 开发能力（Claude Code、Cursor、LangChain）和技术项目经历来弥补

**7. 策略/算法评测经验**
- JD 提到"效果评测"，你有 A/B Test 技能但缺少算法评测的具体案例
- **应对策略**：突出猿辅导的数据分析经验（多数据源采集、转化漏斗）

## 面试准备要点

1. **准备抖音 AI 功能的深度分析**：AI 搜索、AI 生成内容、智能推荐等，能说出优化方向
2. **量化你的 AI 落地成果**：文案自动化（3人→0人）、数据处理（2-3h→分钟级）
3. **展示策略思维**：用转化漏斗、A/B Test 的语言描述你的工作
4. **弱化文科背景，强化技术能力**：主动展示代码能力和 AI 工具链
