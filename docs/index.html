研究交付工作流：前环洞察生产 → 后环可视化交付

前环用 MRW 把 Brief 和研究材料整理成有证据链的洞察报告，后环用 Chart AI 把洞察转成可编辑、可追溯的图表表达。

前环 · 洞察生产

桌面研究洞察报告工作流

Brief / 材料 → 可信洞察
目标人群

产品经理、咨询分析师、行业研究和竞品研究使用者

产品定位

解决研究材料分散、洞察缺少来源支撑的问题，用 Brief 拆解、框架分析和三元校验生成可汇报洞察。
后环 · 表达交付

Chart AI｜研究可视化交付智能体

洞察报告 → 图表表达
目标人群

咨询分析师、产品经理、行业研究和竞品研究使用者

产品定位

解决商业分析从研究材料到可用图表表达慢、口径难追溯的问题，用桌面研究补数、图表模型推荐和结构化校验生成可编辑可视化。
